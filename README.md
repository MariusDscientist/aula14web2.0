# Fundación Aula 14 (A14) - Plataforma Web

Sitio web oficial de la **Fundación Aula 14**, una institución dedicada a transformar el tejido social a través del arte, la cultura y la gestión del patrimonio en Risaralda, Colombia.

## 🎨 Sobre la Fundación
La Fundación Aula 14 (A14) es una entidad sin ánimo de lucro que promueve, fortalece y circula procesos creativos para el desarrollo social y la economía creativa. Nuestra labor abarca desde la luthería y producción musical hasta las artes escénicas y la mediación cultural.

### Áreas de Acción
*   **Formación y Mediación Cultural:** Laboratorios de creación y programas educativos.
*   **Producción Musical y Luthería:** Fabricación y restauración de instrumentos.
*   **Artes Escénicas y Visuales:** Teatro, muralismo y diseño de escenografías.
*   **Gestión y Patrimonio:** Memoria histórica e identidad territorial.

## 🚀 Tecnologías Utilizadas
Este proyecto está construido con un stack moderno para garantizar velocidad y una excelente experiencia de usuario:

*   **React 19:** Biblioteca principal para la interfaz.
*   **Vite:** Herramienta de construcción rápida.
*   **Tailwind CSS 4:** Framework de estilos para un diseño moderno y responsivo.
*   **Lucide React:** Set de iconos minimalistas.
*   **Netlify:** Plataforma de despliegue y CI/CD.

## 🛠️ Instalación y Desarrollo Local

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/TU_USUARIO/fundacion-aula14.git
    cd fundacion-aula14
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    ```

3.  **Iniciar servidor de desarrollo:**
    ```bash
    npm run dev
    ```

4.  **Construir para producción:**
    ```bash
    npm run build
    ```

## 🌐 Despliegue
El sitio está configurado para desplegarse automáticamente en **Netlify** cada vez que se realiza un push a la rama principal. 

*   **Comando de construcción:** `npm run build`
*   **Directorio de publicación:** `dist`
*   **Redirecciones:** Incluye un archivo `public/_redirects` para soportar el enrutamiento de SPAs.
